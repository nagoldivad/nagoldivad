About this website

This website is a static site — i.e., made with only HTML and CSS (and maybe a bit of JavaScript) — hosted on Netlify. It was made by hand using VS Code text editor. The styles and layout were copied/inspired from Papanek by Roman Fink.

A little while ago (May, 2024) I switched from Google Analytics to GoatCounter in order to increase privacy for visitors, and to lighten up on the JavaScript.

In a way this website is my little experiment in creating simplicity on the web. For more on this philosophy, see The small web is beautiful by Ben Hoyt and/or Kagi Small Web.
